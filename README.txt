1º PASSO - Criar uma pasta onde vai ficar o projeto,
Abrir o CMD dentro da pasta e utilizar o seguinte comando para criar o projeto - npx create-expo-app NOME_APP


Estrutura de pastas: 
PACKAGE.JSON - Pasta com as dependências do projeto;
APP.JSON - Dependências do EXPO;
ASSETS - Onde fica as imagens para o utilizar no projeto;
NODE_MODULES - Dependências gerais do projeto.
